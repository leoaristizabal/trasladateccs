Rock Paper Scissors

Este código implementa un juego básico de piedra, papel o tijeras en JavaScript, donde el usuario juega contra la computadora. El código genera una selección aleatoria para la computadora y luego determina quién ganó. El puntaje del usuario y la computadora se actualizan y se muestran en la pantalla.

Weather App
